* Requires

  * ``` python3 ```
  
  * ``` python3-dbus ```
  
  * ``` pygobject3 ```
 
  * ``` python3-gobject ```



* To Use
 
  * ``` cd && git clone https://github.com/yucefsourani/gvrecord ```

  * ``` chmod 755 ~/gvrecord/gvrecord.py ```
  
  * ``` chmod 755 ~/gvrecord/run.sh ```

  * ``` ~/gvrecord/run.sh```



* Install For Fedora

  * ``` sudo  dnf copr enable youssefmsourani/gvrecord  -y ```
  
  * ``` sudo  dnf install gvrecord  -y ```
